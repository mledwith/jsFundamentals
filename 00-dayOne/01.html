First Day Notes

NOTES

  • When using Emmet, class is notated by a "." and id is by "#".
  • HTML stands for hypertext markup language.
  • DOCTYPE states the type of our document. In this case, HTML.
  • our html tag denotes our language to be English, per the lang="en".
  • The head tag houses metadata, or detailed data about our doc.
  • UTF-8 specifies character encoding. Each character (letter, number, even spaces) has a numberic value.
  • Viewport notes the device we are reading our document on.
  • title reflect the name that displays on our browswer tab.
  • link allow us to connect to other files, such as our stylesheet (CSS)
  • astyle tag can help us inject CSS into our document *Not the best practice
  • the body tag houses all the other tags that we want to display onto our browser. This will be what the user sees!
